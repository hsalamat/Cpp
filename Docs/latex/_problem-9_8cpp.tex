\doxysubsection{Detailed Description}
Special Pythagorean triplet A Pythagorean triplet is a set of three natural numbers, a \texorpdfstring{$<$}{<} b \texorpdfstring{$<$}{<} c, for which,. 

a$^\wedge$2 + b$^\wedge$2 = c$^\wedge$2 For example, 3$^\wedge$2 + 4$^\wedge$2 = 9 + 16 = 25 = 5$^\wedge$2.

There exists exactly one Pythagorean triplet for which a + b + c = 1000. Find the product abc.
