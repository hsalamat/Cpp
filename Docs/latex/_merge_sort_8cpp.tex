\doxysubsection{Detailed Description}
merge sort Like Quick\+Sort, Merge Sort is a Divide and Conquer algorithm. 

It divides the input array into two halves, calls itself for the two halves, and then merges the two sorted halves. The \mbox{\hyperlink{_merge_sort_8cpp_a16cb2ba41fd98d8061fea42ea0e99cfa}{merge()}} function is used for merging two halves. The merge(arr, l, m, r) is a key process that assumes that arr\mbox{[}l..m\mbox{]} and arr\mbox{[}m+1..r\mbox{]} are sorted and merges the two sorted sub-\/arrays into one. \begin{DoxyAuthor}{Author}
Hooman Salamat
